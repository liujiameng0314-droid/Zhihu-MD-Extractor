# 📦 ZhihuArchiver (知乎图文归档引擎)

> 一个轻量、高效的知乎问答全自动化抓取与 Markdown 图文归档工具。

## 💡 项目简介

**ZhihuArchiver** 基于底层的浏览器自动化框架 `DrissionPage` 构建，旨在自动进入指定的知乎问题页面，智能突破页面的瀑布流滚动加载限制，将回答正文及高清配图提取出来，并分别导出为本地的 Markdown 文件与 Excel 数据总表。

## ⚠️ 重要前提：账号登录说明 (Read Me First)

受限于知乎严格的反爬虫策略与风控机制，**本程序目前暂不支持“免登录/游客模式”抓取**。在未登录状态下，知乎网页会频繁弹出不可关闭的登录框，导致程序解析失败。

👉 **因此，在正式开始抓取前，请务必确保在程序自动呼出的 Chrome 浏览器中，手动登录你自己的知乎账号。**

## ✨ 核心特性

* **🛡️ 智能向下滚动加载:** 采用“拉扯式”滚动算法（触底后微量回滑），有效触发知乎的异步加载机制，确保所有隐藏回答都被成功刷出。
* **🖼️ Markdown 图文解析:** 自动识别正文中的 `<p>` 段落和 `<figure>` 图片节点，提取 `data-actualsrc` 中的高清原图链接，并转换为标准的 Markdown 图片语法 `![picture](url)`。
* **📂 双格式数据导出:** * **Markdown:** 以答主昵称命名，将每个回答独立保存为 `.md` 文本文件。
    * **Excel:** 自动将所有答主名称及对应回答文本汇总，生成 `data.xlsx` 数据表。
